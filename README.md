# 📊 Marks Average Calculator

A simple web-based application to calculate the average of three subject marks. Built using HTML, CSS, and JavaScript, this project helps students quickly compute their average score in a user-friendly interface.

---

## 🚀 Features

- Input marks for 3 subjects
- Instant average calculation
- Simple and responsive UI
- Beginner-friendly JavaScript logic
- Lightweight and fast (no frameworks used)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
